درس سوم: متغیرها و انواع داده در پایتون

متغیر چیه؟

- متغیر مثل یه جعبه‌ست که می‌تونید اطلاعات (مثل عدد، متن یا چیزای دیگه) رو توش ذخیره کنید.
- تو پایتون، برای ساخت متغیر فقط یه اسم انتخاب می‌کنید و باهاش یه مقدار رو ذخیره می‌کنید. نیازی به تعریف نوعش نیست!
- مثلاً: x = 5 یعنی یه متغیر به اسم x ساختیم که مقدارش ۵ه.

انواع داده در پایتون

داده‌ها تو پایتون چند نوع اصلی دارن:

  • عدد صحیح (int): اعداد بدون اعشار، مثل 10 یا -3.
  • عدد اعشاری (float): اعداد با اعشار، مثل 3.14 یا -0.5.
  • رشته (str): متن‌ها، مثل "سلام" یا "Python". همیشه تو نقل‌قول‌ها ("") نوشته می‌شن.
  • بولین (bool): فقط دو مقدار داره: True (درست) یا False (غلط).
  • لیست (list): مجموعه‌ای از داده‌ها، مثل [1, 2, 3] یا ["سیب", "موز"].

مثال ساده:

بیاید یه برنامه بنویسیم که اسم و سن یه نفر رو بگیره و یه پیام نشون بده:

name = input("اسمت چیه؟ ")
age = int(input("چند سالته؟ "))
print("سلام", name, "تو", age, "ساله هستی!")

تو این کد، name یه رشته‌ست، age یه عدد صحیح و خروجی یه پیام خوشامدگوییه.

نکات مهم:

  • اسم متغیر باید معنی‌دار باشه (مثل age به جای a).
  • پایتون به حروف بزرگ و کوچک حساسه (Name با name فرق داره).
  • می‌تونید نوع داده یه متغیر رو با تابع type() چک کنید، مثلاً print(type(5)) می‌گه <class 'int'>.

جمع‌بندی:

متغیرها و انواع داده ابزارهای اصلی برنامه‌نویسی تو پایتون هستن. با این‌ها می‌تونید اطلاعات رو ذخیره و مدیریت کنید و برنامه‌های جذاب بنویسید!
