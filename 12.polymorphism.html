درس دوازدهم: چندشکلی (Polymorphism) در پایتون

چندشکلی یعنی «چند شکل داشتن» — یعنی یه متد (مثل صدا زدن) تو کلاس‌های مختلف، رفتار متفاوتی داشته باشه!

چندشکلی یعنی چی؟

- چندشکلی یکی از اصول مهم برنامه‌نویسی شیءگراست.
- به ما اجازه می‌ده اشیاء مختلف رو با یه رابط مشترک استفاده کنیم، حتی اگه رفتارشون فرق کنه.
- مثل اینه که به سگ، گربه و پرنده بگی «صدا بده»، هر کدوم یه صدای متفاوت می‌دن!

چندشکلی با ارث‌بری و بازنویسی متدها

وقتی کلاس فرزند متد والد رو بازنویسی (Override) می‌کنه، چندشکلی اتفاق می‌افته:

مثال کاربردی: حیوانات و صدای آن‌ها

بیاید یه برنامه بسازیم که چند حیوان مختلف رو صدا بزنه:

class Animal:
    def speak(self):
        return "من یه حیوانم و صدایی دارم!"

class Dog(Animal):
    def speak(self):
        return "ووف ووف!"

class Cat(Animal):
    def speak(self):
        return "میوووو!"

class Bird(Animal):
    def speak(self):
        return "جیک جیک!"

# ساختن چند شیء
dog = Dog()
cat = Cat()
bird = Bird()

# استفاده از چندشکلی
animals = [dog, cat, bird]

for animal in animals:
    print(animal.speak())

خروجی:

ووف ووف!
میوووو!
جیک جیک!

توجه کنید که همه شیءها متد speak() رو دارن، ولی هر کدوم رفتار خودشون رو اجرا می‌کنن — این یعنی چندشکلی!

چندشکلی با تابع‌های معمولی

حتی بدون کلاس هم می‌تونید چندشکلی داشته باشید! (به این می‌گن Duck Typing)

مثال: تابع که هر چیزی که متد speak داشته باشه رو صدا می‌زنه:

def make_sound(creature):
    print(creature.speak())

# استفاده از تابع با اشیاء مختلف
make_sound(Dog())
make_sound(Cat())
make_sound(Bird())

چندشکلی با متدهای داخلی پایتون

حتی عملگرهایی مثل + و تابع len() هم چندشکلی دارن!

print(len("سلام"))      # خروجی: 4 (تعداد کاراکتر)
print(len(["سیب", "موز"]))  # خروجی: 2 (تعداد عضو لیست)
print("سلام" + " دنیا")   # خروجی: سلام دنیا
print(5 + 3)              # خروجی: 8

همون + و len()، ولی رفتار متفاوت بسته به نوع داده!

مزایای چندشکلی

  • کد تمیزتر و قابل فهم‌تر
  • قابلیت گسترش راحت (می‌تونید کلاس جدید اضافه کنید بدون تغییر کد اصلی)
  • استفاده مجدد از کدها
  • نوشتن توابع عمومی که با انواع مختلف شیء کار کنن

جمع‌بندی:

چندشکلی به ما اجازه می‌ده با اشیاء مختلف، یه جور رفتار کنیم، ولی هر کدوم رفتار خاص خودشون رو داشته باشن. همراه با ارث‌بری، یکی از قوی‌ترین ابزارهای برنامه‌نویسی شیءگرا در پایتونه!
