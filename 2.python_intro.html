درس دوم: معرفی زبان برنامه‌نویسی پایتون

پایتون چیه؟

- پایتون یه زبان برنامه‌نویسی مفسری و سطح بالاست. یعنی کدهاش به زبان انسان نزدیک‌تره و خط به خط اجرا می‌شه.
- مثل یه دوست باهوشه که دستورات ساده شما رو می‌فهمه و سریع اجرا می‌کنه.

ویژگی‌های مهم پایتون:

  • سادگی: کدهای پایتون خیلی خوانا و ساده‌ان. مثلاً برای چاپ "سلام دنیا" فقط می‌نویسیم: 
    print("سلام دنیا")
  • انعطاف‌پذیری: از پایتون می‌تونید برای کارهای مختلف مثل ساخت اپلیکیشن، تحلیل داده یا حتی رباتیک استفاده کنید.
  • کتابخانه‌های زیاد: پایتون کلی کتابخانه داره (مثل NumPy برای محاسبات یا Django برای وب) که کار برنامه‌نویسی رو راحت‌تر می‌کنه.
  • جامعه بزرگ: چون پایتون خیلی محبوبه، منابع آموزشی و انجمن‌های آنلاین زیادی برای کمک به شما وجود داره.

چرا پایتون یاد بگیریم؟

- برای مبتدی‌ها عالیه چون یادگیریش ساده‌ست.
- تو صنعت خیلی استفاده می‌شه، مثلاً تو گوگل، ناسا و اینستاگرام!
- می‌تونید باهاش پروژه‌های واقعی و جذاب بسازید.

مثال ساده:

بیاید یه برنامه بنویسیم که اسم شما رو بگیره و بگه "خوش اومدی!"

name = input("اسمت چیه؟ ")
print("خوش اومدی", name, "!")

این کد از کاربر اسم می‌گیره و یه پیام خوشامدگویی نشون می‌ده.

جمع‌بندی:

پایتون زبانیه که هم برای شروع برنامه‌نویسی عالیه و هم برای پروژه‌های حرفه‌ای. با یادگیری پایتون می‌تونید دنیایی از امکانات رو کشف کنید!
