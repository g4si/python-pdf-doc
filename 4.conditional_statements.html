درس چهارم: ساختار شرطی در پایتون

ساختار شرطی چیه؟

- ساختار شرطی مثل یه تقاطع تو برنامه‌ست که می‌گه: «اگه این شرط درست باشه، این کارو کن، وگرنه یه کار دیگه!»
- تو پایتون، از کلمه‌های if، elif و else برای این کار استفاده می‌کنیم.

چطور کار می‌کنه؟

  • if: اگه شرط درست باشه، کد داخلش اجرا می‌شه.
  • elif: اگه شرط if درست نباشه، این شرط بررسی می‌شه.
  • else: اگه هیچ کدوم از شرط‌های قبلی درست نباشن، این اجرا می‌شه.

ساختار کلی:

if شرط:
    # کاری که اگه شرط درست باشه انجام می‌شه
elif شرط_دیگه:
    # کاری که اگه شرط دوم درست باشه انجام می‌شه
else:
    # کاری که اگه هیچ شرطی درست نباشه انجام می‌شه

مثال ساده:

بیاید یه برنامه بنویسیم که سن یه نفر رو بگیره و بگه می‌تونه رای بده یا نه:

age = int(input("چند سالته؟ "))
if age >= 18:
    print("تو می‌تونی رای بدی!")
elif age >= 16:
    print("هنوز نمی‌تونی رای بدی، ولی نزدیک شدی!")
else:
    print("ببخشید، هنوز خیلی جوونی برای رای دادن!")

این کد سن رو می‌گیره و بر اساس مقدارش پیام مناسب رو نشون می‌ده.

نکات مهم:

  • شرط‌ها باید با عملگرهای مقایسه‌ای نوشته بشن، مثل == (برابره)، != (نابرابره)، >، < و غیره.
  • تورفتگی (فاصله اول خط) تو پایتون خیلی مهمه! کدهای داخل if باید ۴ فاصله (یا یه تب) از اول خط فاصله داشته باشن.
  • می‌تونید از عملگرهای منطقی مثل and، or و not برای ترکیب شرط‌ها استفاده کنید.

مثال پیشرفته‌تر:

برنامه‌ای که بررسی می‌کنه نمره دانش‌آموز چه رتبه‌ای داره:

score = int(input("نمرت رو وارد کن (0-100): "))
if score >= 90:
    print("عالی! رتبه A")
elif score >= 70:
    print("خوب! رتبه B")
elif score >= 50:
    print("قابل قبول! رتبه C")
else:
    print("نیاز به تلاش بیشتر! رتبه D")

جمع‌بندی:

ساختارهای شرطی به برنامه‌هامون اجازه می‌دن هوشمند باشن و بر اساس ورودی‌ها تصمیم‌های متفاوت بگیرن. با تمرین این ساختارها، می‌تونید برنامه‌های خیلی جذاب بنویسید!
