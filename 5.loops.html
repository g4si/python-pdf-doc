درس پنجم: حلقه‌ها در پایتون

سلام بچه‌ها! امروز قراره درباره حلقه‌ها تو پایتون صحبت کنیم. حلقه‌ها به ما کمک می‌کنن یه کار رو چند بار تکرار کنیم بدون اینکه کد رو بارها بنویسیم.

حلقه چیه؟

- حلقه‌ها مثل یه چرخ و فلک تو برنامه‌نویسی‌ان! یه بخش کد رو تا وقتی یه شرط خاص برقراره، تکرار می‌کنن.
- تو پایتون، دو نوع حلقه اصلی داریم: for و while.

حلقه for

- وقتی می‌دونیم یه کار باید چند بار تکرار بشه، از for استفاده می‌کنیم.
- معمولاً با range() یا لیست‌ها کار می‌کنه.

مثال: چاپ اعداد ۱ تا ۵:

for i in range(1, 6):
    print(i)

خروجی: 1 2 3 4 5

حلقه while

- وقتی نمی‌دونیم دقیقاً چند بار قراره یه کار تکرار بشه، ولی یه شرط داریم، از while استفاده می‌کنیم.
- تا وقتی شرط درست باشه، حلقه ادامه پیدا می‌کنه.

مثال: چاپ اعداد تا وقتی کاربر ۰ وارد کنه:

num = int(input("یه عدد وارد کن (۰ برای خروج): "))
while num != 0:
    print("عدد:", num)
    num = int(input("یه عدد دیگه وارد کن (۰ برای خروج): "))

کنترل حلقه‌ها

  • break: حلقه رو کامل متوقف می‌کنه.
  • continue: بقیه کدهای این دور حلقه رو رد می‌کنه و می‌ره دور بعدی.

مثال با break و continue:

for i in range(1, 6):
    if i == 3:
        continue  # عدد ۳ رو رد می‌کنه
    if i == 5:
        break    # حلقه تو ۵ تموم می‌شه
    print(i)

خروجی: 1 2 4

نکات مهم:

  • تورفتگی (۴ فاصله یا یه تب) تو حلقه‌ها خیلی مهمه!
  • حواستون باشه حلقه while اگه شرطش همیشه درست باشه، ممکنه بی‌نهایت اجرا بشه (حلقه بی‌نهایت).
  • برای کار با لیست‌ها، for خیلی کاربردیه، مثلاً: for item in ["سیب", "موز"]:

مثال پیشرفته‌تر:

برنامه‌ای که جمع اعداد ۱ تا ۱۰ رو حساب می‌کنه:

total = 0
for i in range(1, 11):
    total += i
print("جمع اعداد ۱ تا ۱۰:", total)

خروجی: جمع اعداد ۱ تا ۱۰: ۵۵

جمع‌بندی:

حلقه‌ها به ما کمک می‌کنن کارهای تکراری رو راحت‌تر انجام بدیم. for برای تعداد مشخص و while برای شرط‌های خاص عالی‌ان. با تمرین، می‌تونید برنامه‌های پیچیده‌تری بنویسید!
