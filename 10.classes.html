درس نهم: کلاس‌ها در پایتون

کلاس چیه؟

- کلاس یه راه برای تعریف یه نوع داده جدیده که شامل داده‌ها (ویژگی‌ها) و توابع (متدها) مربوط به اونه.
- مثل یه نقشه برای ساختن یه ماشین: کلاس مشخص می‌کنه ماشین چه ویژگی‌هایی (مثل رنگ) و چه کارهایی (مثل حرکت) داره.
- با کلمه class تعریف می‌شه.

چطور کلاس بسازیم؟

ساختار کلی یه کلاس:

class Name:
    def __init__(self, parameters):
        # ویژگی‌ها (attributes)
        self.parameter = parameter
    
    def method(self):
        # کارهایی که کلاس می‌تونه انجام بده
        pass

- __init__: متد سازنده‌ست که وقتی یه شیء از کلاس ساخته می‌شه، اجرا می‌شه.
- self: به خود شیء اشاره داره و برای دسترسی به ویژگی‌ها و متدها استفاده می‌شه.

مثال ساده:

بیاید یه کلاس برای دانش‌آموز بسازیم:

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"من {self.name} هستم و {self.age} سالمه!"

# ساختن شیء از کلاس
student1 = Student("علی", 15)
print(student1.introduce())

خروجی: من علی هستم و 15 سالمه!

ویژگی‌ها و متدها

  • ویژگی‌ها (Attributes): داده‌های مربوط به شیء، مثل name و age.
  • متدها (Methods): توابعی که کار خاصی برای شیء انجام می‌دن، مثل introduce.

مثال پیشرفته‌تر:

کلاسی که نمره‌های دانش‌آموز رو مدیریت می‌کنه:

class Student:
    def __init__(self, name, grades):
        self.name = name
        self.grades = grades
    
    def calculate_average(self):
        total = sum(self.grades)
        return total / len(self.grades)
    
    def passed(self):
        return self.calculate_average() >= 10

# ساختن شیء
student1 = Student("مریم", [18, 15, 19])
print(f"نام: {student1.name}")
print(f"میانگین: {student1.calculate_average()}")
print(f"قبول شده؟ {'بله' if student1.passed() else 'خیر'}")

نکات مهم:

  • هر شیء یه نمونه (instance) از کلاسه و ویژگی‌های خودش رو داره.
  • می‌تونید چند شیء از یه کلاس بسازید، مثلاً student2 = Student("رضا", 16).
  • کلاس‌ها برای برنامه‌نویسی شیءگرا (OOP) استفاده می‌شن که باعث می‌شه کد منظم‌تر و قابل استفاده مجدد باشه.
  • تورفتگی داخل کلاس (۴ فاصله) خیلی مهمه!

جمع‌بندی:

کلاس‌ها به ما اجازه می‌دن اشیاء با ویژگی‌ها و رفتارهای مشخص بسازیم. با کلاس‌ها می‌تونید برنامه‌های پیچیده‌تر و سازمان‌یافته‌تر بنویسید، مثل یه سیستم مدیریت دانش‌آموز!
