بازگشت به فهرست مطالب

درس هفتم: توابع در پایتون

تابع چیه؟

- تابع یه بلوک کدِ که یه کار خاص رو انجام می‌ده و هر وقت لازمش داشتید می‌تونید صداش کنید.
- تو پایتون با کلمه def تعریف می‌شه و می‌تونه ورودی بگیره، کاری انجام بده و خروجی بده.
- مثلاً: یه تابع برای گفتن "سلام" به یه نفر.

چطور تابع بسازیم؟

ساختار کلی یه تابع:

def function_name(پارامترها):
    # کدهایی که تابع انجام می‌ده
    return خروجی  # اختیاری

- def: شروع تعریف تابع.
- پارامترها: ورودی‌هایی که تابع می‌گیره (اختیاری).
- return: مقداری که تابع برمی‌گردونه (اختیاری).

مثال ساده:

بیاید یه تابع بسازیم که به کاربر سلام کنه:

def say_hello(name):
    message = "سلام " + name + "!"
    return message

# صدا زدن تابع
print(say_hello("علی"))

خروجی: سلام علی!

انواع تابع

  • بدون پارامتر: مثل def greet(): print("hello!")
  • با پارامتر: مثل مثال بالا که name پارامتره.
  • با مقدار پیش‌فرض: می‌تونید برای پارامترها مقدار پیش‌فرض بذارید.
  • با بازگشت: با return مقدار برمی‌گردونه.

مثال با پارامتر پیش‌فرض:

تابعی که جمع دو عدد رو حساب می‌کنه:

def add_numbers(a, b=0):
    return a + b

print(add_numbers(5, 3))  # خروجی: 8
print(add_numbers(5))     # خروجی: 5 (چون b=0)

نکات مهم:

  • اسم تابع باید معنی‌دار باشه، مثل calculate_average.
  • تورفتگی داخل تابع (۴ فاصله) خیلی مهمه!
  • با return می‌تونید نتیجه رو به برنامه اصلی برگردونید، وگرنه تابع None برمی‌گردونه.
  • توابع باعث می‌شن کدتون تمیزتر و قابل استفاده مجدد باشه.

مثال پیشرفته‌تر:

تابعی که میانگین یه لیست از نمره‌ها رو حساب می‌کنه:

def calculate_average(scores):
    total = 0
    for score in scores:
        total += score
    average = total / len(scores)
    return average

# صدا زدن تابع
grades = [85, 90, 78, 92]
print("میانگین نمره‌ها:", calculate_average(grades))

خروجی: میانگین نمره‌ها: 86.25

جمع‌بندی:

توابع ابزارهایی هستن که کد رو منظم‌تر و قابل استفاده مجدد می‌کنن. با توابع می‌تونید برنامه‌های پیچیده‌تری بنویسید و کارها رو ساده‌تر کنید!
