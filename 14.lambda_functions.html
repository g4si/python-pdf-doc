درس چهاردهم: توابع lambda در پایتون

تابع lambda — یعنی تابع بدون اسم!

تابع lambda چیه؟

- lambda یه راه سریع برای ساختن تابع‌های کوچیک و یک‌خطی هست.
- به جای اینکه با def یه تابع کامل بنویسیم، با کلمه lambda یه تابع موقت می‌سازیم.
- معمولاً وقتی استفاده می‌شه که یه تابع ساده فقط یک بار نیاز داریم (مثلاً داخل sorted یا map).

ساختار تابع lambda

lambda ورودی‌ها : خروجی

مثال:

# تابع معمولی
def square(x):
    return x * x

# همون تابع با lambda
square_lambda = lambda x: x * x

print(square(5))         # خروجی: 25
print(square_lambda(5))  # خروجی: 25

کاربردهای اصلی lambda

1. استفاده با sorted() — مرتب‌سازی سفارشی

مرتب کردن لیست دانش‌آموزان بر اساس نمره:

students = [("علی", 17), ("مریم", 19.5), ("رضا", 15), ("سارا", 18)]

# مرتب کردن بر اساس نمره (عنصر دوم هر تاپل)
sorted_students = sorted(students, key=lambda student: student[1])

print(sorted_students)
# خروجی: [('رضا', 15), ('علی', 17), ('سارا', 18), ('مریم', 19.5)]

2. استفاده با map() — اعمال تابع روی همه اعضای لیست

دو برابر کردن همه اعداد یه لیست:

numbers = [1, 2, 3, 4, 5]

doubled = list(map(lambda x: x * 2, numbers))
print(doubled)  # خروجی: [2, 4, 6, 8, 10]

3. استفاده با filter() — فیلتر کردن اعضای لیست

انتخاب فقط اعداد زوج:

numbers = [1, 2, 3, 4, 5, 6, 7, 8]

evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # خروجی: [2, 4, 6, 8]

4. lambda با چند ورودی

# جمع دو عدد
add = lambda x, y: x + y
print(add(5, 3))  # خروجی: 8

# بررسی اینکه آیا اسم با "ا" شروع می‌شه؟
starts_with_a = lambda name: name.startswith("ا")
print(starts_with_a("احمد"))   # True
print(starts_with_a("رضا"))    # False

مزایای تابع lambda

  • کد کوتاه‌تر و تمیزتر می‌شه
  • برای توابع ساده و یک‌بار مصرف عالیه
  • همراه با توابع map، filter، sorted خیلی قدرتمنده

معایب و نکات مهم

  • فقط یک عبارت (expression) می‌تونه داشته باشه — دستورات مثل print یا if چندخطی نمی‌تونه باشه
  • برای توابع پیچیده بهتره از def استفاده کنید (خواناتری داره)
  • زیاده‌روی نکنید! گاهی تابع معمولی بهتر و قابل‌فهم‌تره

جمع‌بندی

تابع lambda مثل یه ابزار جادویی کوچیکه که وقتی نیاز به یه تابع ساده و سریع دارید، به کمکتون میاد. مخصوصاً تو مرتب‌سازی، فیلتر کردن و تبدیل داده‌ها، استفاده از lambda کد شما رو حرفه‌ای و کوتاه‌تر می‌کنه!
